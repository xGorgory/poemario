# Versos del Umbral — Poemario Personal

Un poemario digital minimalista y elegante diseñado para capturar pensamientos, versos y sentires de forma íntima.

## ✨ Características

- 🌙 **Modo Oscuro Nativo**: Diseñado para la lectura nocturna y la introspección.
- 💾 **Persistencia Local**: Tus poemas se guardan en tu navegador (`localStorage`).
- 🏷️ **Categorización**: Filtra tus versos por temáticas (Amor, Melancolía, Tiempo, etc.).
- 📖 **Experiencia de Lectura Inmersiva**: Navegación fluida entre poemas y tipografía cuidadosamente seleccionada.
- 🔒 **Acceso Protegido**: Panel de administración protegido por PIN para añadir o eliminar versos.
- 🎨 **Estética Premium**: Partículas sutiles, cursor personalizado y animaciones de revelado.
- 🖨️ **Modo Impresión**: Estilos optimizados para exportar tus poemas a papel o PDF.

## 🚀 Cómo usar

Simplemente abre `index.html` en cualquier navegador moderno.

### Administración
Para añadir o borrar poemas:
1. Haz clic en **+ Añadir**.
2. Introduce el PIN por defecto: `5010`.
3. Una vez autenticado, podrás añadir nuevos versos o borrarlos desde la vista de lectura.

## 🛠️ Tecnologías

- HTML5 Semántico
- CSS3 (Vanilla con variables y animaciones)
- JavaScript (Vanilla, estructura modular)
- Google Fonts (EB Garamond, Cormorant Garamond, Marcellus)
